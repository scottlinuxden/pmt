Project Action Item (PAI)
User's Manual

System Requirements
Any operating system which can execute either Netscape 4.0+ or Internet Explorer 4.0+.  JavaScript must be enabled within your browser which is the default.  Cookies should be enabled as well.  An Internet connection, dial-up or dedicated and access to the Institute for Software Research, Projects And Research Park website.  This web will transfer information via an SSL web server therefore your web browser must support secure server connections through certificates.

Login Instructions
Before you can execute any PAI functions you must first login.  To login you must enter a valid username and password combination.  A username and password may be acquired by submitting e-mail to support@isrparc.org.

To login to PAI you must select Action Items from the Site Menu bar in the left frame of your browser.  A login screen will appear in the right frame of your web browser.  To login enter your username and password that have been provided to you and depress the Query button.  You may also depress the Help button to access this user's manual.

Action Item Listing
Once you have logged in you will be presented with a listing of all Action Items in the PAI database.  You may at this point select either Edit, Delete, or View.  Only those project members with the authority to delete will have the Delete button available for selection.

Configuring the Action Item Listing
The Action Item listing may contain a variety of information from each action item.  The listing is configured by selecting the field selector above each column on the listing.  To display a different action item field under the column field selector, select the drop down list on Windows platforms or the option menu selector on Unix platforms.  All possible fields that are viewable for an Action Item are presented for selection.  Select the field of interest and depress the Query button.  You must depress the Query button before the field data for the field selected is displayed in the Action Item listing.  You may depress the Query button under the "Add Column" area or at the bottom of the Action Item listing.

Adding Fields to the Action Item Listing
Additional fields may be added to the Action Item listing.  This is done by selecting the "Add Column" option drop down on Windows platforms or the option popup menu on Unix platforms.  Select the field you would like added to the listing and depress the Query button.  The Action Item listing will be re-displayed with the additional field presented on the right most part of the list.  For instance, to view only those action items which have not been completed, select the "Add Column" drop down, select 'Completion Date' then depress the Query button.  The Action Item listing will be re-displayed will all completion dates shown for each action item.  You should then select '=' then enter 'NULL'  in the criteria textbox, see the section Criteria Area, under the field selector, and depress the Query button.  Only those action items that have not been completed will be displayed.

Criteria Area
The Criteria Area contains a relational operator (equal, not equal, less than, greater than, etc) drop down and a criteria textbox to provide criteria that each action item must meet before it is displayed in the Action Item listing.  For instance, to designate that only action items assigned to 'Davis' and 'Not Completed' be displayed select the equals operator under the 'Assigned To' field selector and enter 'Davis' in the criteria textbox.  Select the equals operator under the 'Completion Date' field selector and enter 'NULL' in the criteria textbox.  Do not enter the single quotes in the criteria textbox.  Note, you must currently have the 'Assigned To' and 'Completion Date' field selectors already displayed if they are not already by using the 'Add Column' drop down.  Select the Query button once these steps have been performed.

Creating an Action Item
To create an action item simply depress the Create button at the bottom of the Action Item Listing.  An blank Action Item Form will be presented for entry.  Entry all required information and depress the Save button.

Editing an Action Item
Editing an action item is quite simple.  Just depress the Edit button on the same row as the action item to edit.  The Action Item form will be displayed for editing.  Enter the information on the form that is available to you and is required, fields designated in bold and red, and depress the Save button.  Note: the data you enter is not saved until you depress the Save button.  Hitting the Back button on your web browser or leaving the form in anyway will not save the form.  Options presented on the form include Save, Delete (if authorized), View, Listing, and Help.  As mentioned earlier the Save button will save the action item, Delete will delete the action item, View will present a read-only display of the action item and is ideal for printing the action item,  Listing will display the Action Item Listing.  Help will present this user's manual.

Deleting an Action Item
Action Items may be deleted from the Action Item Listing or the Action Item Form.  A confirmation box will appear asking you to confirm the deletion.  If the Ok button is depressed the Action Item will be deleted and can not be recovered.  If you depress the Cancel button the deletion is cancelled and the Action Item is retained.

Viewing and Action Item
Viewing an Action Item is primarily to support viewing a read-only version of the action item.  The Action Item Form is displayed in a more compact manner and is presented in a an ideal formatting for printing from your web browser.

Printing an Action Item
Select the View button from the Action Item Listing or Action Item Form.  Select 'Print Frame' from your web browser after selecting the Action Item Form.

Operator Alerts
An Operator Alerts Area is presented on the Action Item Listing and Action Item Form.  This area should be monitored to verify that the operation that has been performed was executed successfully or failed.

Bug Reports
Submit all bug reports for PAI to System Problem Report Tracking Tool or if you are not an authorized user send an e-mail.
